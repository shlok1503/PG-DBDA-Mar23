# Ridge Regression:
	-make use of L2 Regularization
	-perform feature weight updates as a loss function has an additional square term

# Lasso Regression:
	-make use of L1 Regularization
	-perform feature weight updates as a loss function has an additional square term
	containing the L1 norm of the weights vector
